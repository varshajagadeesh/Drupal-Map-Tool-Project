# Secure Location Map

To run the complete local website on Windows, double-click:

```text
START-HERE.cmd
```

The first start installs Drupal and imports all four supplied CSV files automatically.

Read [HOW-TO-RUN.md](HOW-TO-RUN.md) for the beginner guide, website addresses, login, stopping, resetting, and troubleshooting.

The Drupal custom module is located at:

```text
web/modules/custom/secure_location_map
```
